# v2board机场用户流量监控机器人

本项目旨在帮助机场主揪出恶意消耗流量的坏蛋

# 前言
目前只能运行在[Google Apps Script]([url](https://www.google.com/script/start/)https://www.google.com/script/start/), 主要是方便
后期会添加nodejs, php脚本, 感兴趣的同学也可以一起维护这个库
